# Hacienda Los Aromos

Mockup y sitio web de Hacienda Los Aromos — un proyecto elegante y profesional.

## Característica

- Diseño limpio y moderno
- Interfaz intuitiva con navegación tabular
- Paleta de colores personalizados
- Totalmente responsivo

## Tecnología

- HTML5
- CSS3 vanila

## Cómo usar

1. Abre `index.html` en tu navegador
2. Navega por las diferentes secciones usando las pestañas

## Deploy

Este proyecto está desplegado en Vercel y se actualiza automáticamente con cada push a la rama `main`.
